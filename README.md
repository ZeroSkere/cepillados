# 🍧 Sistema de Gestión de Cepillados

Sistema administrativo para control de inventario, pedidos y ventas de cepillados.

## 🚀 Características

- 📊 Dashboard con estadísticas en tiempo real
- 📝 Registro de pedidos con despacho rápido
- 📸 Imágenes de sabores para publicidad
- 💰 Múltiples métodos de pago
- 📈 Reportes con gráficos
- 🌙 Modo claro/oscuro
- 📱 Diseño responsive
- 📄 Exportación a Excel y PDF
- 💱 Tasa BCV automática

## 📋 Requisitos

- Python 3.8+
- pip
